# Lección 3: Funciones

## ¿Qué es una Función?

Una función es un bloque de código reutilizable que realiza una tarea específica.

### Definición básica:
```python
def nombre_funcion(parametros):
    # código
    return resultado
```

## Componentes de una Función

1. **def**: Palabra clave para definir una función
2. **nombre_funcion**: Nombre descriptivo de la función
3. **parametros**: Valores de entrada (opcional)
4. **return**: Devuelve un valor (opcional)

## Tipos de Parámetros

- **Parámetros posicionales**: Se pasan por orden
- **Parámetros con nombre**: Se especifica el nombre
- **Parámetros por defecto**: Tienen un valor predeterminado
- **Args y kwargs**: Número variable de argumentos

## Scope (Alcance)

- **Variables locales**: Solo existen dentro de la función
- **Variables globales**: Accesibles en todo el programa
