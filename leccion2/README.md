# Lección 2: Estructuras de Control

## Condicionales

Los condicionales permiten ejecutar código basado en condiciones.

### if, elif, else
```python
if condicion:
    # código si la condición es verdadera
elif otra_condicion:
    # código si la otra condición es verdadera
else:
    # código si ninguna condición es verdadera
```

## Bucles

Los bucles permiten ejecutar código repetidamente.

### Bucle for
Se usa para iterar sobre secuencias (listas, strings, rangos, etc.)
```python
for elemento in secuencia:
    # código a ejecutar
```

### Bucle while
Se ejecuta mientras una condición sea verdadera
```python
while condicion:
    # código a ejecutar
```

## Control de Flujo

- `break`: Sale del bucle inmediatamente
- `continue`: Salta a la siguiente iteración
- `pass`: No hace nada (placeholder)
