# Lección 1: Introducción a Python

## Variables y Tipos de Datos

En Python, una variable es un contenedor para almacenar valores. No necesitas declarar el tipo de variable explícitamente.

### Tipos de datos básicos:
- **int**: Números enteros (1, 2, 3, -5)
- **float**: Números decimales (3.14, -0.5)
- **str**: Cadenas de texto ("Hola", 'Python')
- **bool**: Valores booleanos (True, False)

## Operadores

### Operadores aritméticos:
- `+` : Suma
- `-` : Resta
- `*` : Multiplicación
- `/` : División
- `//` : División entera
- `%` : Módulo (resto)
- `**` : Potencia

### Operadores de comparación:
- `==` : Igual a
- `!=` : Diferente de
- `>` : Mayor que
- `<` : Menor que
- `>=` : Mayor o igual que
- `<=` : Menor o igual que

## Entrada y Salida

- `print()`: Muestra información en la consola
- `input()`: Recibe entrada del usuario
