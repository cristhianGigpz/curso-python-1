# Ejercicios del Curso de Python

Este directorio contiene ejercicios prácticos para cada lección del curso.

## Instrucciones

1. Lee el enunciado de cada ejercicio
2. Intenta resolver el problema por tu cuenta
3. Ejecuta tu código para verificar que funciona
4. Compara con las soluciones proporcionadas (si están disponibles)
