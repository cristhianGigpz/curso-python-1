# Lección 4: Estructuras de Datos

## Listas

Las listas son colecciones ordenadas y mutables (modificables).

```python
mi_lista = [1, 2, 3, 4, 5]
```

**Métodos importantes:**
- `append()`: Agregar al final
- `insert()`: Insertar en posición
- `remove()`: Eliminar elemento
- `pop()`: Eliminar y retornar
- `sort()`: Ordenar

## Tuplas

Las tuplas son colecciones ordenadas e inmutables (no modificables).

```python
mi_tupla = (1, 2, 3, 4, 5)
```

## Diccionarios

Los diccionarios almacenan pares clave-valor.

```python
mi_dict = {"nombre": "Ana", "edad": 25}
```

**Métodos importantes:**
- `keys()`: Obtener claves
- `values()`: Obtener valores
- `items()`: Obtener pares clave-valor
- `get()`: Obtener valor de forma segura

## Conjuntos (Sets)

Los conjuntos son colecciones no ordenadas de elementos únicos.

```python
mi_set = {1, 2, 3, 4, 5}
```

**Operaciones:**
- `union()`: Unión de conjuntos
- `intersection()`: Intersección
- `difference()`: Diferencia
